MY COMMENTS ON THE PERSONALITY TEST

The 16 personalities test can be used to get a general overview of one's personality, however one should not rely entirely on this test to know their personality type.
It is accurate in the sense that I did it twice and got the same results so that proves its reliabilty but please note that only licensed clinical psychologists
have specialisted training in administering these type of assessments and tests and interpreting the results. Therefore, the 16 personalities test is
slightly valid but if you need a completely accurate result, contact a clinical psychologist.
